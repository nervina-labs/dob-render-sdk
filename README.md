## How to run the examples

0. Build the sdk

```bash
pnpm run build
```

1. Change to the browser example directory

```bash
cd examples/browser-env
```

2. Install dependencies

```bash
pnpm i
```

3. Run the example

```bash
pnpm run dev
```
